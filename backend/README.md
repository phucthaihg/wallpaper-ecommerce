# Wallpaper E-commerce Backend

A Node.js backend for a wallpaper and home decor e-commerce platform.

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- Sequelize ORM
- JWT Authentication

## Prerequisites

- Node.js (version 14 or higher)
- PostgreSQL
- npm
